# Wireframes

Put annotated Zoho CRM screenshots and your own sketches here, organized
by module.

Suggested files (create as you go):

```
wireframes/
├── leads-list.png
├── leads-list.notes.md
├── leads-detail.png
├── leads-detail.notes.md
├── deals-kanban.png
├── deals-kanban.notes.md
├── ...
```

Each `*.notes.md` should answer:
- What's the user trying to accomplish on this screen?
- Which fields/columns/widgets are essential vs. nice-to-have?
- Any RBAC notes (who sees what)?
- Any empty-state, loading-state, error-state notes?

Reference these from your `agent-os/specs/<spec>/spec.md` so the agent
has visual context when planning.
