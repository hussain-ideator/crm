# Components

Shared, app-wide UI primitives — used across more than one feature.

- `components/ui/` is where shadcn/ui primitives land when added via
  `npx shadcn add <name>`. These are copy-paste components owned in this repo;
  edit them freely.
- Anything specific to a single feature belongs in that feature's folder
  (`features/<resource>/components/`), not here.
